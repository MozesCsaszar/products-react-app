## Project Info

This is a webpage that displays products and their reviews. It uses React.js, Material UI and TypeScript for the frontend with Node.js for the backend.

Backend provided by [this](https://github.com/tonymihut/rw-api) repository.

## Setup

### Backend

See `./backend/README.md`.

### Frontend

1. Navigate to the frontend folder:

```bash
cd ./frontend/
```

2. Install dependencies:

```bash
npm install
```

3. Create .env file

```.env
BASE_URL=<your_backend_url>
```

4. Run the development server:

```bash
npm run dev
```

You will be able to access the frontend server at [http://localhost:5000/](http://localhost:5000/).
